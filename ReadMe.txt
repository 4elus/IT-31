Папка: assets\css - В этой папке находится bootstrap разметка (bootstrap.min / bootstrap-theme.min ) , которая позволяет делать адаптивную вёрстку. (Сайт: https://getbootstrap.com/ )
файл: slick.css - С помощью данного файла задаются стили слайдера
файл: style.css - В этом файле мы описываем все наши стили.


Папка: assets\fonts 
В этой папке находится шрифты GoogleFonts (Сайт: https://fonts.google.com/ ) 

Папка: assets\img
В этой папке храняться изображения.

Папка: assets\js

bootstrap.min.js - Храниться ужатая версия всех необходимых функций, например: Выпадающие меню.

slick.js - С помощью данного файла работает слайдер.

script.js - В этом файле мы описываем весь наш JS код.

header.html и footer.html  -  подключается к .php файлам с помощью метода include('name.html') 

index.php - является основной страницей сайта.
